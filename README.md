 
These 4 projects train your brain to instantly recognize:

> “Is the model too dumb (high bias / underfitting) or too sensitive (high variance / overfitting)?”

All projects use simple polynomial regression + NumPy / scikit-learn / Matplotlib.

## Projects Overview

| # | Project Name                        | Core Lesson                                          | Key Visual / Output                              |
|---|-------------------------------------|------------------------------------------------------|--------------------------------------------------|
| 1 | Model Complexity Ladder             | Bias ↓ & Variance ↑ as complexity increases          | U-shaped test error curve                        |
| 2 | Dataset Sensitivity Test            | Variance = instability across similar datasets       | Two very different fits on almost identical data |
| 3 | Bootstrap Instability Visualizer    | High variance = wide spread of bootstrap predictions | Shaded ±1 std band around mean prediction        |
| 4 | Underfit vs Overfit Playground      | Recognize failure patterns from train/val error gap  | Side-by-side underfit (straight line) vs overfit (wiggly) plots |
